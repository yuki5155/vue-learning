# TODOアプリ ドキュメント

## 概要
このTODOアプリは、Vue.jsとTypeScriptを使用して構築されたシングルページアプリケーションです。ユーザーはタスクの追加、編集、削除、完了のマーキングなどの基本的なTODO管理機能を利用できます。

## 技術スタック
- Vue.js 3
- TypeScript
- Vue Router
- Vuex
- CSS (ネイティブ)

## 始め方
```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build
```
